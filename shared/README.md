# Shared — Cross-Platform Assets & Documentation

This directory contains all assets, brand guidelines, and documentation that are shared across the admin dashboard and customer mobile app.

## Structure

```
shared/
├── assets/
│   ├── icons/          — SVG icon set
│   └── logos/          — Ramyas Jewellers logo variants
├── branding/
│   ├── color-palette/  — Brand color definitions
│   ├── typography/     — Font choices and scale
│   └── component-guidelines/ — UI component usage rules
├── docs/
│   ├── api/            — API endpoint documentation
│   ├── database/       — Database schema documentation
│   └── design-system/  — Design system reference
├── constants/          — Shared constants (e.g., scheme rules)
└── types/              — Shared type definitions (JSON Schema / OpenAPI)
```
