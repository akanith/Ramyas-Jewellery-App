# Ramyas Jewellers — Typography System

## Font Families

| App | Primary Font | Secondary Font | Monospace |
|---|---|---|---|
| Admin Dashboard (Web) | Inter | — | JetBrains Mono |
| Customer App (Mobile) | Poppins | — | — |

## Web Typography Scale (Inter)

| Token | Size | Weight | Line Height | Usage |
|---|---|---|---|---|
| `display-xl` | 48px | 700 | 1.1 | Hero headings |
| `display-lg` | 36px | 700 | 1.2 | Page titles |
| `heading-xl` | 30px | 600 | 1.3 | Section headings |
| `heading-lg` | 24px | 600 | 1.3 | Card titles |
| `heading-md` | 20px | 600 | 1.4 | Sub-section titles |
| `heading-sm` | 18px | 500 | 1.4 | Label headings |
| `body-lg` | 16px | 400 | 1.6 | Primary body text |
| `body-md` | 14px | 400 | 1.6 | Secondary body text |
| `body-sm` | 12px | 400 | 1.5 | Captions, hints |
| `label-md` | 14px | 500 | 1.4 | Form labels, nav |
| `label-sm` | 12px | 500 | 1.4 | Badges, chips |

## Mobile Typography Scale (Poppins)

| Token | Size | Weight | Usage |
|---|---|---|---|
| `headlineLarge` | 28sp | 700 | Screen titles |
| `headlineMedium` | 24sp | 600 | Section headers |
| `titleLarge` | 20sp | 600 | Card titles |
| `titleMedium` | 16sp | 600 | List item titles |
| `bodyLarge` | 16sp | 400 | Primary body |
| `bodyMedium` | 14sp | 400 | Secondary body |
| `bodySmall` | 12sp | 400 | Captions |
| `labelLarge` | 14sp | 500 | Buttons |
| `labelSmall` | 11sp | 500 | Tags, badges |

## Guidelines

- Use weight 600+ for all interactive UI elements
- Avoid using more than 2 type sizes per screen section
- Ensure all text meets WCAG AA contrast minimums
